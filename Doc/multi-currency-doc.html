2 — Architektur-Prinzip

Systemwährung: EUR. Alle Beträge werden bei Buchung in EUR konvertiert und gespeichert. Das Wallet läuft immer in EUR. Lehrer setzen ihren Preis in ihrer Währung. Schüler wählen ihre Anzeigewährung.

Best Practice (wie Stripe, Airbnb, Booking.com): Eine einzige Systemwährung im Backend. Konvertierung nur an der UI-Grenze. Kurs wird zum Buchungszeitpunkt eingefroren und im Slot gespeichert. 
/* Geldfluss-Beispiel */
Lehrer (GEL):  Preis = ₾130  (Lari)
               Kurs  = 1 EUR = 2.965 GEL
               → slot.price = 43.85 EUR  ← gespeichert in EUR

Schüler (USD): Wallet = $500
               Sieht:  $47.36  (43.85 EUR × 1.08)
               Zahlt:  $47.36 vom Wallet (in USD-Anzeige)
               Wallet intern: -43.85 EUR

3 — Provider Swap-Mechanismus

Der einzige Unterschied zwischen Mockup und Production ist eine Datei.

ModusDatei ladenVerhalten

Kein weiterer Code ändert sich.

4 — CurrencyService API

/* Konvertierung (async) */
CurrencyService.convert(amount, from, to, function(err, result) {
  // result: converted amount in target currency
});

/* Konvertierung (sync, nur wenn Kurse gecacht) */
var eur = CurrencyService.convertSync(45, 'GEL', 'EUR');

/* Formatierung mit Symbol */
CurrencyService.format(45.00, 'EUR');  // → "€45,00"
CurrencyService.format(130, 'GEL');    // → "₾130,00"
CurrencyService.format(5000, 'JPY');   // → "¥5000"
CurrencyService.format(45, 'SEK');    // → "45,00 kr"

/* Symbol */
CurrencyService.getSymbol('AED');  // → "د.إ"

/* Alle Währungen */
CurrencyService.getSupportedCurrencies();
// → [{code:'EUR', symbol:'€', name:'Euro', flag:'🇪🇺', decimals:2}, ...]

/* Mockup-Check */
CurrencyService.isMock();  // → true in Mockup, false in Production

/* Warmup (beim App-Start) */
CurrencyService.warmUp(function(err, rates) { ... });

/* Profil-Währung lesen */
CurrencyService.getTeacherPriceCurrency('t1');  // → 'GEL'
CurrencyService.getUserCurrency('s1');          // → 'USD'

5 — Konvertierungs-Pipeline (bei Buchung)

1
Lehrer hat Preis 130 GEL in Profil (priceCurrency: 'GEL')
↓
2
Schüler klickt „Buchen" → adapter-localstorage.bookSlot()
↓
3
CurrencyService.convertSync(130, 'GEL', 'EUR') → 43.85 EUR
↓
4
slot.price = 43.85 — in EUR gespeichert. Kurs eingefroren.
↓
5
Escrow: fullAmount = 43.85 EUR. Deposit: 8.77 EUR (20%)
↓
6
UI zeigt dem Lehrer: ₾130,00 · dem Schüler (USD): $47.36
