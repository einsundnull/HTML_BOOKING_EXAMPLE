3 — ProfileStore API

Storage: localStorage['app_profiles'] — JSON-Objekt, Key = UID.

ProfileStore.get(uid) → profile | null
Gibt gespeichertes Profil zurück oder null wenn keines existiert.
ProfileStore.getDefault(uid) → profile
Erstellt Profil-Objekt mit allen Defaults. Liest Store.Users.byUid(uid) für name/email.
ProfileStore.getOrDefault(uid) → profile
Shorthand: get(uid) || getDefault(uid). Immer sicher aufzurufen.
ProfileStore.save(uid, profileData) → profileData
Speichert Profil in localStorage. Setzt automatisch uid und updatedAt.
ProfileStore.getDisplayName(uid) → string
Gibt profile.name zurück falls vorhanden, sonst user.name, sonst uid.
ProfileStore.getPhoto(uid) → string | null
Gibt profile.photo zurück oder null. Genutzt von buildAvatarHTML().
ProfileStore.getPrice(uid) → string | null
Gibt profile.pricePerHalfHour oder null. Genutzt in Katalog und Buchungsflow.
ProfileStore.getSpecializations(uid) → string[]
Gibt profile.specializations oder [].
ProfileStore.getSkillLevel(uid) → string NEU
Gibt profile.skillLevel ('beginner'|'intermediate'|'advanced'|'expert') oder ''. Nur für Schüler relevant.

Öffnet animiertes Bottom-Sheet mit Profildaten eines beliebigen Nutzers. Zeigt: Avatar, Name, Rollen-Badge, Standort, Stats (skillLevel, Erfahrung, Preis), Sprachen, Bio. Footer-Link → profile-view.html?uid=&viewer=.
