Timezone Day View — Bug Analysis

Why changing timezone shifts the start of the slot list · 2026-03-24

Root Cause Summary

The 24h grid and the slot lookup are mismatched.

The grid allTimes[] is always built as ["00:00", "00:30", ... "23:30"] — 48 UTC strings, always starting at 00:00 UTC.

Slots are stored as UTC. slotMap[t2] looks up by UTC key — correct.

But _tdvBuildRow(slotMap[t2], t2, dateStr) converts t2 (UTC) → local for display only.

Result: the grid always starts at 00:00 UTC and ends at 23:30 UTC. If teacher is UTC+5, their "real" working day starts at 05:00 UTC. The first 10 slots (00:00–04:30 UTC = previous night locally) appear at the top of the list with local times like "05:00–09:30". Everything appears shifted.

What the fix must do

#FixWhereHow
1 Build allTimes grid in local order teacher.js renderDayPanel() Build 48 UTC times sorted by their local equivalent. Start from local midnight (= UTC midnight minus offset), wrap around. So UTC+5 grid starts at UTC 19:00 (= local 00:00) and ends at UTC 18:30 (= local 23:30).
2 Section labels use local time ranges teacher.js renderDayPanel() Convert section boundaries (Nacht 00:00–05:30) to UTC equivalents using teacher's TZ before comparing to t2. Or compare _localTime against local section boundaries instead.
3 Fetch slots from adjacent dates at boundary adapter-localstorage.js getSlotsByTeacherDate() For non-UTC teachers, also fetch slots from prev/next UTC date and include ones whose local date matches dateStr. Needed for UTC+10 or UTC-10 edge cases.

/* Do: build UTC times ordered by local start */
var teacherTZ     = TimezoneService.getUserTimezone(currentUser.uid);
var offsetMinutes = TimezoneService._getOffsetMinutes(teacherTZ, dateStr); /* need to expose this */
var localMidnightUtcMin = (0 - offsetMinutes + 1440) % 1440; /* UTC minutes = local 00:00 */

/* Build 48 slots starting from local midnight in UTC */
var allTimes = [];
for (var i = 0; i < 48; i++) {
  var utcMin = (localMidnightUtcMin + i * 30) % 1440;
  var h = Math.floor(utcMin / 60);
  var m = utcMin % 60;
  allTimes.push((h<10?'0':'')+h+':'+(m<10?'0':'')+m);
}
