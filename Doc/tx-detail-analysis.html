4 — Was gebaut werden muss
FeatureBeschreibung
TX-Detail-Sheet Klick auf TX-Item öffnet ein Bottom-Sheet / Modal mit allen Feldern aus Tabellen 1+2. Layout: Header (Typ-Badge + Betrag), Body (alle Meta-Felder in Label:Wert-Paaren), Footer (TX-ID für Support).
Icons + Labels 4 fehlende TX-Typen (booking, move, teacher_cancel, lesson_confirmed) in TX_ICONS und wallet.json nachtragen.
Description ungekürzt Im List-Item description auf ~60 Zeichen kürzen mit "…", im Detail vollständig zeigen.
Zeitspanne bei Block-TX Bei booking-TX mit meta.slotCount > 1: Uhrzeit als "14:00–15:30 (3 Slots)" anzeigen.
