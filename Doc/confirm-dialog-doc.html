3 — Betragsberechnung

SzenarioFreizugeben jetztFormel
Deposit bezahlt (held) + instant fullAmount - depositAmount Deposit bereits abgezogen → nur Rest freigeben
Kein Deposit / deposit unpaid fullAmount Voller Betrag wird bei Bestätigung abgezogen
Bar vor Ort (cash_on_site) 0 Kein Wallet-Abzug — Zahlung erfolgt direkt
Kein Escrow vorhanden 0 Fallback — kein Zahlungsblock angezeigt
