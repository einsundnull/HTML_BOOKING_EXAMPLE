Positionierungs-Algorithmus
dashboard.js — _insertNowLine(slotsCol, blocks)
var now = _currentTimeStr(); // "14:22" var insertAfterIndex = -2; // -2 = vor allen, -1 = nicht zeigen for (var i = 0; i < blocks.length; i++) { if (now < blocks[i].start) { insertAfterIndex = i - 1; // vor diesem Block break; } if (now >= blocks[i].start && now < blocks[i].end) { insertAfterIndex = i; // nach diesem Block (läuft gerade) break; } insertAfterIndex = -1; // alle vorbei → kein Line } if (insertAfterIndex === -1) return; // kein Line if (insertAfterIndex === -2) { slotsCol.insertBefore(lineEl, slotsCol.firstChild); // vor allen } else { slotsCol.insertBefore(lineEl, cardEls[insertAfterIndex].nextSibling); }
now vs. BlöckeinsertAfterIndexErgebnis
now < block[0].start-2Linie vor der ersten Karte
now zwischen block[0] und block[1]0Linie nach Karte 0
now innerhalb block[1]1Linie nach Karte 1 (läuft)
now >= block[N-1].end-1Kein Line — alle vorbei
Zeitvergleich: Strings im Format "HH:MM" werden lexikographisch verglichen — "14:22" < "14:30" ist true. Das funktioniert korrekt für alle Uhrzeiten im Format 00:00–23:59.

Aufrufkette
1
dashboard.js — _buildSchedule()
for each dateStr === _dashToday
Erkennt heutigen Tag. Setzt slotsCol.id = 'dash-today-slots' und slotsCol._dashBlocks = blocks — beides für den Ticker benötigt.
2
dashboard.js
_insertNowLine(slotsCol, blocks)
Berechnet Einfügeposition. Erstellt .dash-now-line-Element via _buildNowLineEl(timeStr). Fügt es zwischen den Flex-Kinder-Karten ein.
3
dashboard.js
_startNowLineTick()
setInterval(fn, 60000) — jede Minute. Findet #dash-today-slots, liest ._dashBlocks, ruft _insertNowLine() erneut auf. Alte Linie wird entfernt, neue eingefügt.
4
dashboard.js — _rerender()
_nowLineInterval = null → clearInterval
Bei _rerender() wird der Interval durch _startNowLineTick() neu gestartet — das bestehende Interval wird mit clearInterval(_nowLineInterval) zuerst gestoppt, kein Memory-Leak.
