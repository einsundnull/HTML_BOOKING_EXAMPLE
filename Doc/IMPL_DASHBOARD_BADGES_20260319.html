Schlüsselregel: canMove = !isPast && !isConfirmed — Verlegen und Stornieren sind nur erlaubt solange die Stunde weder vergangen noch bestätigt ist. Bestätigte Stunden können nur über „Freigeben" (Lehrer) wieder geöffnet werden.

dashboard.js — _buildBlockCard & _openBlockDialog
/* State-Bestimmung — identisch in Karte und Dialog */ var badgeState; if (isPast && isConfirmed) badgeState = 'done'; else if (isPast && !isConfirmed) badgeState = 'overdue'; else if (isConfirmed) badgeState = 'confirmed'; else badgeState = 'pending'; /* Aktions-Gating */ var canConfirm = !isPast && !isConfirmed; var canMove = !isPast && !isConfirmed; var canCancel = !isPast && !isConfirmed; var canRelease = !isPast && isConfirmed && currentUser.role === 'teacher';
