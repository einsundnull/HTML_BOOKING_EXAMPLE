ZWECK:
  Tagesansicht eines einzelnen Tages aus dem Kalender heraus öffnen.
  Zeigt alle Zeitslots (06:00–22:00, 30-Min-Intervalle) vertikal,
  horizontal vollflächig, mit Status und optionalen Infos pro Slot.

──────────────────────────────────────────────────────────────────
SLOT ROW — DETAILSTRUKTUR (horizontal, ~72px hoch)
──────────────────────────────────────────────────────────────────

┌──────────┬──────────────────────────────────────┬──────────────┐
│  ZEIT    │  INFO-BEREICH (flex: 1)               │  AKTION      │
│          │                                       │              │
│  08:00   │  [Pill: Verfügbar]                    │   [+]        │
│  –08:30  │                                       │              │
│  (mono,  │  oder:                                │  (runder     │
│  small)  │  [Avatar] Max Mustermann              │   Button,    │
│          │           max@mail.de · 08:00–08:30   │   optional)  │
│          │                                       │              │
└──────────┴──────────────────────────────────────┴──────────────┘

  Spalte 1 – ZEIT:
    · Breite: ~72px, fix
    · Startzeit groß (14px, semibold, monospace)
    · Endzeit klein darunter (12px, neutral-500)
    · Linker Rand: farbige Linie (3px) je nach Status

  Spalte 2 – INFO:
    · Status "available":   Pill "Verfügbar" (grün/navy)
    · Status "booked":      Name + E-Mail des Schülers (2 Zeilen)
    · Status "timeout":     Pill "Gesperrt" (amber)
    · Status "recurring":   Pill "Wiederkehrend" (navy-outline)
    · Status "empty":       Keine Info, nur Grauton

  Spalte 3 – AKTION:
    · Optional, nur wenn Slot aktionsfähig ist
    · [+] für leere Slots anlegen
    · [→] für gebuchte Slots Details öffnen
    · [✕] für timeout aufheben


──────────────────────────────────────────────────────────────────
STATUS-FARBEN (konsistent mit bestehendem Design-System)
──────────────────────────────────────────────────────────────────

  available  →  Linker Rand: --color-400 (navy-blau)
                BG:          --color-50
                Pill:        --color-100 / --color-700

  booked     →  Linker Rand: --color-900 (dunkel-navy)
                BG:          --color-900 (dunkel)
                Text:        --neutral-0 (weiß)

  timeout    →  Linker Rand: --pending-amber
                BG:          --slot-timeout-bg
                Pill:        --slot-timeout-tx

  recurring  →  Linker Rand: --color-400, gestrichelt
                BG:          --color-50
                Pill:        navy-outline

  empty      →  Linker Rand: --neutral-200
                BG:          --neutral-0 (weiß)
                Text:        --neutral-300


──────────────────────────────────────────────────────────────────
ÖFFNEN / SCHLIESSEN
──────────────────────────────────────────────────────────────────

  Öffnen:
    · Klick auf einen Tag im Monatskalender
      (statt bisheriges Day-Slots-Panel seitlich)
    · Overlay slides in von unten (transform: translateY)

  Schließen:
    · [←] Zurück-Button → schließt Overlay, kehrt zu Kalender zurück
    · Swipe-down (mobile, optional Phase 2)


──────────────────────────────────────────────────────────────────
RESPONSIVE
──────────────────────────────────────────────────────────────────

  Mobile  (< 480px):  Slot-Höhe 64px, Zeit-Spalte 60px
  Tablet  (480–768px): Slot-Höhe 72px, Zeit-Spalte 72px
  Desktop (> 768px):  Slot-Höhe 80px, Zeit-Spalte 80px
  Topbar: immer fullwidth, Datum zentriert
