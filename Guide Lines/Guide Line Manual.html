Color Scale System (50–900)

Die Zahlen wie 400, 600, 900 sind keine Farben, sondern Positionen auf einer Helligkeitsskala derselben Farbe.

---

Grundprinzip

WertBedeutung
50sehr hell (fast weiß)
100hell
200leicht hell
300disabled / grau
400Standard / Accent
500Hover
600Active
700–900sehr dunkel bis fast schwarz
---

Beispiel (HSL basiert)

--color-400: hsl(210, 60%, 50%);
--color-500: hsl(210, 60%, 40%);
--color-600: hsl(210, 60%, 30%);

👉 Nur die Lightness wird verändert.

---

UI Verwendung

Use CaseColor Level
Background50–100
Disabled300
Button Default400
Hover500
Active600
Dark UI / Text700–900
---

Interaktion Beispiel

.button {
    background: var(--color-400);
}

.button:hover {
    background: var(--color-500);
}

.button:active {
    background: var(--color-600);
}
---

Wichtige Regeln

  • Nur eine Farbe (Hue) verwenden
  • Alle Varianten über Helligkeit steuern
  • Keine zufälligen Hex-Farben
  • Keine Gradients verwenden
  • Immer Design Tokens nutzen
---

Merksatz

50 = hell → 900 = dunkel

400 = Standard, 500 = Hover, 600 = Active
